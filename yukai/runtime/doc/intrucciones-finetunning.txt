Busco optimizar la compresión y el formato de respuesta de una IA ante preguntas sobre un historial médico de un paciente.

El formato que quiero para los ejemplos es json:
{
    "context": ...
    "prompt": ...
    "response": ...
}

El context es fijo. Y el prompt tiene un formato estructurado.
El context, especifica el rol y las instrucciones para decofificar el formato estructurado del prompt y es el siguiente:

En la lista de documentos cada documento está separado del anterior por ||. Por ejemplo documento 1||documento 2||documento 3
Cada documento con una visista del paciente y está compuesto por una serie de campos, pueden estar todos o no. Cada campo se codifica como n.valor. Campos múltiples separados por |. Listas separadas por ;.Delimitadores internos reemplazados por ¬.Fin de documento ||. Mapeo:0:identificador del documento,1=fecha documento,2=motivo del documento,5=síntomas,6=estado físico,7=medicación,8=tratamiento,9=recomendaciones,10=ingresos,11=comentarios,12=diagnósticos,13=antecedentes familiares,14=factores riesgo cardiovascular,15=alergias,16=operaciones,17=implantes,18=otros
Debes responder a la pregunta únicamente basándote en el contenido de los documentos y de los datos del paciente. No añandas ni infieras nada que no esté en los documentos. Por cada dato que incluyas en la respuesta indica el campo 0 del documento en el que has encontrado la información con la siguiente forma: doc-ref[identificador del documento]
